# 🔀 Condicionais em JavaScript

Esta seção reúne os exercícios desenvolvidos durante meus estudos sobre **estruturas condicionais em JavaScript**.

O objetivo foi aprender como um programa pode tomar decisões com base em diferentes condições, permitindo criar aplicações mais inteligentes e interativas.

---

# 🎯 Objetivo

Praticar o uso de estruturas condicionais para controlar o fluxo de execução de um programa e resolver problemas utilizando lógica de programação.

---

# 📚 Conceitos estudados

Durante os exercícios pratiquei os seguintes conceitos:

- Estrutura `if`
- Estrutura `else`
- Estrutura `else if`
- Operadores de comparação
- Operadores lógicos
- Condições simples e compostas
- Aninhamento de condicionais
- Validação de entradas do usuário

---

# 🛠️ Operadores utilizados

## Operadores de comparação

| Operador | Significado            |
| -------- | ---------------------- |
| `==`     | Igual                  |
| `===`    | Estritamente igual     |
| `!=`     | Diferente              |
| `!==`    | Estritamente diferente |
| `>`      | Maior que              |
| `<`      | Menor que              |
| `>=`     | Maior ou igual         |
| `<=`     | Menor ou igual         |

---

## Operadores lógicos

| Operador | Função    |     |         |
| -------- | --------- | --- | ------- |
| `&&`     | E (AND)   |     |         |
| `        |           | `   | Ou (OR) |
| `!`      | Não (NOT) |     |         |

# 💡 Desafios encontrados

Durante os exercícios percebi a importância de compreender:

- A diferença entre `==` e `===`;
- O funcionamento dos operadores `&&` e `||`;
- O fato de que `prompt()` retorna uma **string**, exigindo conversão quando necessário;
- A organização da lógica para evitar condições desnecessárias ou repetidas.

Esses desafios contribuíram para fortalecer meu raciocínio lógico e melhorar a forma como estruturo meus programas.
