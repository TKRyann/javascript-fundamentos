# 🧠 O que aprendi

## ✅ Estrutura do `for`

Aprendi que o `for` reúne em uma única linha três partes importantes:

- Inicialização
- Condição
- Incremento

Exemplo:

```javascript
for (let i = 1; i <= 10; i++) {
  console.log(i);
}
```

Essa estrutura torna o código mais organizado quando já sabemos quantas vezes o bloco será executado.

# 🔍 Diferença entre `for` e `while`

Compreendi que:

### `for`

É mais indicado quando sabemos quantas vezes a repetição acontecerá.

Exemplos:

- Contagens
- Tabuadas
- Percorrer listas
- Somatórios

### `while`

É mais indicado quando não sabemos quantas repetições serão necessárias.

Exemplos:

- Sistema de login
- Validação de senha
- Menus interativos
- Jogos
