# 📚 Conteúdo estudado

## ✅ Variáveis

Aprendi a:

- Declarar variáveis utilizando `let`
- Armazenar diferentes tipos de informações
- Alterar valores de variáveis
- Utilizar variáveis para montar mensagens dinâmicas

Exemplos praticados:

- Cadastro de usuário
- Informações de produtos
- Dados de filmes
- Time favorito
- Profissão

---

## ✅ Entrada e saída de dados

Pratiquei:

- `prompt()`
- `alert()`
- `console.log()`

Utilizando esses recursos para receber informações do usuário e exibir resultados.

---

## ✅ Operações matemáticas

Exercícios envolvendo:

- Soma
- Subtração
- Multiplicação
- Divisão
- Dobro de um número
- Antecessor e sucessor
- Média de notas
- Conversão de horas para minutos
- Cálculo aproximado de idade em dias

---

## ✅ Manipulação de texto

Aprendi a construir mensagens utilizando:

- Concatenação
- Template Strings
- Variáveis dentro de frases

Exemplo:

```javascript
let nome = "Carlos";
console.log(`Olá, ${nome}!`);
```
