# 🧠 O que aprendi

## ✅ O que é uma função

Aprendi que uma função é um bloco de código criado para realizar uma tarefa específica.

Depois de criada, ela pode ser chamada sempre que necessário.

Exemplo:

```javascript
function mensagem() {
  console.log("Olá, mundo!");
}

mensagem();
```

# ⚠️ Diferença entre `console.log()` e `return`

Uma das principais diferenças aprendidas foi:

### `console.log()`

Serve para mostrar informações.

```javascript
console.log("Olá");
```

---

### `return`

Serve para devolver um valor.

```javascript
return numero * 2;
```

O valor retornado pode ser armazenado em uma variável, utilizado em cálculos ou passado para outras funções.
